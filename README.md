# Sandbox Skills

Public Codex skills for AL sandbox workflows.

## AL Sites

Install the AL Sites skill from `skills/al-sites`.

```text
从 git@github.com:ShanyouYu-Sean/sandbox-skills.git 的 skills/al-sites 安装 al-sites skill
```

The skill configures the remote AL Sites MCP endpoint and verifies the health endpoint. Restart Codex or open a new session after MCP configuration so the `k8s_e2b_sites` tools are loaded.
